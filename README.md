# im-infra

Command line tool for managing docker-cloud infrastructure and deployment

```bash
im-infra -f
im-infra up

im-infra down

im-infra deploy

im-infra scale node -n 3

# not sure if this should stay in the docker-cloud.yml
im-infra scale app -n 2
```
